## 📝 தேர்வுக்கான பயிற்சி உத்திகள்

### ✅ தினசரி பயிற்சி முறை
- **2 மணிநேரப் பயிற்சி**: ஒவ்வொரு நாளும் 2 மணிநேரம்
  - முதல் 1 மணி: கோட்பாடு கற்றல்
  - இரண்டாம் 1 மணி: செயல்முறைப் பயிற்சி

### 📊 வாராந்திர மதிப்பீடு
- ஒவ்வொரு வார இறுதியிலும் 100 வினாக்கள் தேர்வு
- தவறான விடைகளைப் பகுப்பாய்வு செய்தல்
- பலவீனமான பகுதிகளைக் கண்டறிந்து மேம்படுத்தல்

### 📅 மாதாந்திர மீள்பார்வை
- ஒவ்வொரு மாத இறுதியிலும் முந்தைய மாதப் பாடங்களுக்கான முழுத்தேர்வு
- 200+ வினாக்கள் கொண்ட மாதாந்திரத் தேர்வு

### 📒 குறிப்புகள் தயாரித்தல்
- ஒவ்வொரு அலகுக்கும் சுருக்கமான குறிப்புகள் தயாரித்தல்
- முக்கியமான விதிகள், சொற்கள், குறள்கள் ஆகியவற்றுக்குத் தனித்தனிப் பட்டியல்

### 🔍 பிழைப் பகுப்பாய்வு
- ஒவ்வொரு தேர்விலும் தவறான விடைகளைப் பகுப்பாய்வு செய்தல்
- பிழைகளுக்கான காரணங்களை ஆய்வு செய்தல்
- மீண்டும் மீண்டும் பயிற்சி செய்தல்
